# minesweeper_bot

slackとかでマインスイーパが楽しめます（たぶん）

### 導入

例によってhubotで作成したプロジェクトのscriptsフォルダにminesweeper.coffeeを入れてください

### コマンド

@(bot名) start   :盤面を初期化し、ゲームを開始します。リトライする場合もこのコマンドを送ってください。
@(bot名) (座標)    :ゲーム開始後、開けるマスを指定します。座標はa1のように縦横で指定してください。
